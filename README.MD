# APRENDIENDO PYTHON DESDE 0 
En el presente repositorio se plasmarán las actividades y rutas de aprendizaje para convertirme de una persona normal a un Python Dev.

Originalmente, la carpeta llamada "Aprendiendo_Python" es una bóveda del programa OBSIDIAN, el cual es ampliamente utilizado para estos casos de aprendizaje.

## Cómo usar este repositorio en Obsidian

Este repositorio está optimizado para ser leído directamente en GitHub o mediante **Obsidian**.

Si deseas abrirlo como una bóveda local:
1. Clona o descarga este repositorio.
2. Abre Obsidian y selecciona **"Open folder as vault"** (Abrir carpeta como bóveda).
3. Selecciona la carpeta raíz de este proyecto. 
4. *Nota: Las configuraciones de la interfaz e historial están ignoradas en el `.gitignore` para no interferir con tus preferencias personales de Obsidian.*
